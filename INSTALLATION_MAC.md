# Installation du Companion sur Mac

1. Conserve le dossier complet du projet dans un emplacement stable, idéalement `Documents/Tesla Charge Companion`.
2. Ouvre `companion/Tesla Companion.app` avec clic droit → **Ouvrir** au premier lancement.
3. Tu peux faire glisser l’application dans le Dock, mais ne déplace pas l’application seule hors du dossier du projet : elle utilise les fichiers voisins.
4. GitHub Desktop et Firefox doivent être installés dans le dossier Applications.

## Mise à jour Tesla

Le Companion ouvre Firefox sur `about:debugging` et le guide. Charge :

`firefox_extension/manifest.json`

Puis utilise l’extension **Tesla Charge Companion Updater**.

## Sauvegardes

Le bouton **Créer une sauvegarde** copie les fichiers JSON dans :

`backups/AAAA-MM-JJ_HH-MM-SS/`
