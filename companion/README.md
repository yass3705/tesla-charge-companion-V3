# Tesla Companion V5.0

Double-clique sur `Tesla Companion.app`.

Le Companion permet de :

- ouvrir la procédure de mise à jour Tesla dans Firefox ;
- mettre à jour les devises localement ;
- sauvegarder les fichiers JSON ;
- ouvrir GitHub Desktop ;
- ouvrir le site ;
- afficher le dossier du projet.

La lecture automatisée des pages Tesla reste effectuée par l’extension Firefox, car Tesla bloque les scripts Python, GitHub Actions et Playwright.
