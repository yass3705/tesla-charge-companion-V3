# Mise à jour Tesla

1. Double-clique sur `companion/Tesla Companion.app`.
2. Choisis **Mettre à jour Tesla avec Firefox**.
3. Dans Firefox, clique sur **Charger un module complémentaire temporaire**.
4. Sélectionne `firefox_extension/manifest.json`.
5. Clique sur l’icône de l’extension.
6. Sélectionne `data/tesla_stations.json`.
7. Clique sur **Démarrer**.
8. Laisse Firefox traiter les stations.
9. À la fin, remplace `data/tesla_stations.json` par le fichier téléchargé.
10. Ouvre Tesla Companion et choisis **Ouvrir GitHub Desktop**.
11. Summary : `Mise à jour Tesla`.
12. Clique sur **Commit to main**, puis **Push origin**.

Cette procédure reste nécessaire car Tesla bloque les méthodes d’automatisation classiques.
