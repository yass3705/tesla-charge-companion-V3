# Premier lancement — V5.0

## 1. Remplacer les fichiers du dépôt local

1. Décompresse `Tesla_Charge_Companion_V5_0.zip`.
2. Dans GitHub Desktop, ouvre ton dépôt.
3. Menu **Repository → Show in Finder**.
4. Copie tout le contenu du dossier V5.0 dans le dossier du dépôt.
5. Accepte le remplacement.

## 2. Publier la V5

Dans GitHub Desktop :

1. Summary : `Passage à la V5.0`
2. Clique sur **Commit to main**.
3. Clique sur **Push origin**.
4. Attends la publication automatique GitHub Pages.

## 3. Ouvrir Tesla Companion

1. Ouvre le dossier `companion`.
2. Fais un clic droit sur `Tesla Companion.app`.
3. Clique sur **Ouvrir**.
4. Confirme avec **Ouvrir** si macOS affiche un avertissement.

Le menu du Companion apparaît.
